# Hedge Simulation

## Setup Instructions

1. **Clone the repository**  
   ```sh
   git clone https://github.com/bachmannscode/quant_hedging
   ```

2. **Change into the project directory**  
   ```sh
   cd quant_hedging
   ```

3. **Install dependencies**
   ```sh
   python3 -m venv venv
   . venv/bin/activate
   python3 -m pip install -r requirements.txt
   ```

4. **Run the simulation**
   
   Run `simulation.py`.
